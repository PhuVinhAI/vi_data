# Report phân tích từ vựng: nhìn (Rank 103)

## 1. Thông tin chung
- **Từ:** nhìn
- **Rank:** 103
- **Level:** A1 (Bậc 1)
- **Topic:** Core Verbs

## 2. Phân tích nội dung
- **Định nghĩa:** "look / see". Động từ chỉ hành động quan sát.
- **Ví dụ:** "Nhìn vào bức tranh", "nhìn ra cửa sổ", "nhìn thấy gì". 
- **Quizzes:** "Tôi ___ thấy bức tranh đẹp." -> "nhìn".
- **Ghi chú (Notes):** Phân biệt sắc thái giữa "nhìn" (hành động nhìn) và "nhìn thấy" (kết quả nhìn thấy/nhận ra). So sánh với "xem" (watch) giúp người học dùng từ chính xác hơn.

## 3. Đánh giá tính phù hợp
- Dữ liệu cơ bản, đầy đủ, hướng dẫn cách dùng đi kèm giới từ chỉ hướng (vào, ra, lên, xuống).
