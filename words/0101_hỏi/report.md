# Report phân tích từ vựng: hỏi (Rank 101)

## 1. Thông tin chung
- **Từ:** hỏi
- **Rank:** 101
- **Level:** A1 (Bậc 1)
- **Topic:** Core Verbs

## 2. Phân tích nội dung
- **Định nghĩa:** "ask / inquire".
- **Ví dụ:** "Muốn hỏi về bài học", "hỏi anh về kế hoạch", "đừng ngại hỏi". 
- **Quizzes:** "Tôi muốn ___ về bài học." -> "hỏi".
- **Ghi chú (Notes):** Hướng dẫn dùng "hỏi + người + về + việc". Nhắc đến nghĩa mở rộng như xin phép (ask permission). So sánh với cụm từ trang trọng "đặt câu hỏi".

## 3. Đánh giá tính phù hợp
